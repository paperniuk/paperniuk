## 👋 About Me

I am a System Integration & Automation Engineer working at the intersection of
backend engineering, artificial intelligence, business automation and applied
machine learning research.

My engineering work focuses on transforming fragmented manual workflows into
structured, scalable and maintainable software systems.

I build:

- backend services and REST APIs;
- ERP and CRM automation;
- system integrations and data pipelines;
- web scraping and reporting tools;
- AI agents and LLM-assisted workflows;
- applied machine learning and computer vision systems.

Alongside engineering, I work on university-level AI research involving
Physics-Informed Neural Networks, deepfake detection, image authenticity
analysis and machine learning model evaluation.
